## 2024.08.10
现在已是8月11日的凌晨了。长久以来，我的写作欲在不断降低以至于近乎消失，两次写作之间的间隔，有时是周计，有时是月计，有时甚至是半年计。上次写作已经是十天前的事情了，究其原因我可以说是太忙，生活循规蹈矩没有新意，但其实以前喜欢写东西的时候，生活也是这样，但并不会减低我的写作兴趣，事无巨细地记录下一切。而现在我确实更加觉得这千篇一律的生活没有任何记录的必要，更多地是由于我不知从何时开始的对任何事物都没有兴致的悲剧所导致。我开始急躁不安，没有耐心，对大部分情绪化的输入表现出没有任何情绪，自然也就没有想进行记录这种本身需要很多情绪和感触所进行的活动。算起来最初的停笔是2021年8月，之后将近6个月后的2022年2月写了一些文字后，又是漫长的断笔，大概足足有18个月后的8月又开始了寥寥几笔的记录，这中间发生了太多的事情，而这些事情完全改变了我的心态。在那之后便不再有长篇连续的记录，我不再在乎每天的事情都要详细地有迹可循，好像这个事情不再重要。

但是我还是想说说这段时间发生的事情，重要的事情不多，可能也就是8月4日美丽的夕阳，四妹总是再念叨她那永远无法到达的新车，七夕的今天与首富和昆仑的相聚，依然如往昔般，我带着大家聊他的家庭现状，房子这些没有营养的话题，我知道我们不会聊那些触及灵魂的东西，仅仅是在一块聚聚就已经很好了。我依然记得2020.08.08那天，以及这一天的谐音所代表的含义，当我度过2024.08.08时候所发生的感伤。

而，七夕的今天我依然像平常一样度过，一来如上文所述我不再对任何事情抱有兴趣，所以对四妹的表达也就没有很多，二来我本是特立独行之人，别人做的事情我是绝对不大想去做的，那么今日大家都会做的事情对我来说就是反感之极，虽然四妹对我有所不满，但她更多说的就是对新车的不满。我寄了一些补品给了母亲，我想七夕所做之事可能就是与二人同聚而已，私以为这一天最应做的就是在星空下踟蹰感受天琴与天鹰的寂寞与怅惘，而不是那些蝇营狗苟之事，那些pyq传递出的不正之风，故明天夜爬穹窿，坐观英仙，吾辈义不容辞。

我想对于我来说，能赋予我勇气来应对任何人任何事的挑战的，就是在这里一直记录下去的这件事本身吧。那些视频或许也应该继续了。

我不再注重于每天花多少钱，仔细记录，每天想了什么，每天发生了什么，女朋友每天做了什么，说了什么，对什么东西好奇，甚至于她为什么生气，喜欢什么也不想记录，甚至于过段时间能完全忘记，这到底是为什么呢。我与学生时代的我到底有什么区别，那些夜晚喜欢遐想的我到底去了哪里。

我并不是每天都在记录，但是转而又会想如果不是时刻记录，多年以后回想，这段时光的细节荡然无存，就比如2023年的4月春天之于我到底有什么重要意义的事情呢，但是又会想人是不可能无时无刻都在记录的，文字无法承载一切，或许我应该用相机去记录生活，从而慰藉我现在这种对于时光逝去的恐惧。

逐渐丧失的表达欲...不想记录，不想发图，不想思考。何止于此，我想与年龄，这段时光的经历，那件事等有关吧。但是，如此这般便会感觉仿佛并没有在真正的生活，每天便如行尸走肉，若干年后的今天也不会回想起今天昨天做了什么。想像那个人一样记录读书心得，以及那个人一样记录爱情，以及那个人一样记录思考。以前总是在做流水账，其实并不会去回看，如今想记录一些有意义的东西，如果什么都没有，流水账是没有意义的。譬如，照片，譬如，故事，譬如，感悟，诸如此类。从2021的8月开始断笔，到如今已经过去了28个月，中间在2022年2月写了一篇文章后，便再无下文，这中间寥寥几笔便可概括。毕业，工作，再恋爱，母亲。

# 2022年
## 2022年2月
### 2022.2.5
&emsp;&emsp;我已经疲于设置那些花哨的格式，炫酷的样式，乱七八糟的标签。我说过不再写日记了，但这不是日记，就像它的名字一样，这是一个魔咒，无论是我给自己下的还是来自于他人的无心，我已然困顿在这宥匣之内，无法逃脱。
</br></br>
2021年，已经结束了，从各种意义上讲。这是我人生中最糟糕的一年，我想也许还要加个之一，毕竟以后的人生只能更糟，但对于以往的人生来说，最字还是用上的。想来我已经停笔了近160多天，五个多月，这瞬间的每分每秒我都在变得更加抑郁，颓废，绝望。身体状态不断下降，心好像困在了《秒五》中的那辆载着贵树去见明里的火车上，孤独地穿梭于黑夜，不断地跳动，不断地前行，下不下车已然不重要，继续着这场没有尽头的旅途。
</br></br>
我由听见变成听不见，我由乐观变为悲观，我由短暂快乐变为无尽痛苦，我由爱一切沦为连自己都不爱的废物。我的生命好像已经过了半年，可是我意识与精神仿佛静止般，什么也表达不出来了，我好像是一个无尽无聊的生物体，支撑我的好像只有生命的本能。
</br></br>
说实话，我已经体会不到真正的快乐是什么样的了。无论是吃到好吃的美食，还是玩了好玩的游戏，还是见到好朋友，还是跟有意思的人聊了很久的天，还是过节过年...任何事情都显得没有任何意义，我好像会笑，但是我已经不会快乐了。
</br></br>
